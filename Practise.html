Практический Этап

HTML состоит из ряда элементов, которые используются для того, чтобы обернуть или разместить различные части содержимого, чтобы оно имело определенный вид. Встроенные тэги могут преобразовать часть содержимого в гиперссылку, выделить курсивом слова и так далее. Можно рассмотреть следующую строку:

Создание Сайта

Если мы хотим чтобы строка никак не менялась и отобразиласть также на сайте, то можно определить ее как “Параграф”, заключив в теги элемента

< p> Создание Сайта < p>

Основные Части Элемента
Материал из Developer.Mozilla.org
CSS – это код, для стилизации вашей веб-страницы. Например, чтобы выбрать все элементы “параграфа” на странице и изменить цвет текста на красный, вы можете написать следующие строчки:
CSS

p {

color: red;

}

А чтобы выбрать определенный элемент в HTML файле, то нужно присвоить ему ID.

< p id=”название” > Создание Сайта < p >

В CSS это Будет Выглядеть так
CSS

# название {

color: red;

}
